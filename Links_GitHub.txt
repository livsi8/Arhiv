Что бы подготовиться прошу: 
1) создать свой аккаунт на https://github.com/
2) Скачать и установить https://desktop.github.com/ Вариантов github-ов достаточно много. Этот вариант мне больше нравится.

git config --global user.name "Solovyov Oleg"
git config --global user.email "livsi8@gmail.com"
git config --list

Command examples:
git init //Инициализация нового проэкта
git add Old // Добавление в локальный github папки Old
git add .  // Добавление в локальный github всех папок
git cd Old //Перейти в подкаталог Old
git status //Покажет ошибки, например недобавленные в локальный github папки.
git commit -m "Изменил ... что-то там" //Добавляем комментарий к файлам и папкам в локальном github.
git push -u origin master //Спросит логин и пароль для синхронизации локального github с сервером (без добавления комментария добавление на сервер не произойдет).
